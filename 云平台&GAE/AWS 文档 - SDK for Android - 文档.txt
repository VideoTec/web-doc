AWS Mobile SDK for Android

Welcome to the AWS Mobile SDK for Android Developer Guide. 
The AWS Mobile SDK for Android is an open-source software development kit, 
distributed under an Apache Open Source license. 
The SDK for Android provides 
1）libraries, 
2）code samples, and 
3）documentation to help developers build connected mobile applications using AWS. 

Supported AWS services currently include:

1）Amazon Cognito
http://aws.amazon.com/cognito

2）Amazon Mobile Analytics
http://aws.amazon.com/mobileanalytics/

3）Amazon S3
http://aws.amazon.com/s3/

4）Amazon DynamoDB
http://aws.amazon.com/dynamodb/

5）Amazon Kinesis
http://aws.amazon.com/kinesis

6）AWS Lambda
http://aws.amazon.com/lambda

This guide will help you start developing Android applications using Amazon Web Services. 
If you’re new to the AWS Mobile SDK, 
you’ll probably want to look first at What is the AWS SDK for Android? 
http://docs.aws.amazon.com/mobile/sdkforandroid/developerguide/Welcome.html
and Set Up the AWS Mobile SDK for Android. 
http://docs.aws.amazon.com/mobile/sdkforandroid/developerguide/setup.html
These topics explain what the AWS Mobile SDK includes and 
how to set up the SDK in an Android development environment.

After you’ve set up the SDK, 
you can start working with the mobile clients for Amazon Web Services.
