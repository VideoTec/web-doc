Telephony software Skype uses hole punching 
to allow users to communicate with one or more users audibly.

Fast-paced online multi-player games may use a hole punching technique or 
require users to create a permanent firewall pinhole in order to reduce network latency.

VPN application Hamachi utilizes hole punching 
to allow users to connect directly to subscribed devices behind firewalls.

Decentralized peer-to-peer file sharing software 
relies on hole punching for file distribution.
